pwd

January 1, 2016

pwd is short for Print (current) Working Directory.

用来干嘛

顾名思义就是用来打印出当前所在的目录名字。

Usage

1/ 不接任何选项

2/ With -P, --physical

假设当前目录是一个指向一个其他目录的符号链接(symlink)，那么pwd默认打印出的目录名字是这个符号链接的名字，使用-P选项可以打印出最终指向的「物理」路径。

More

在Bash中，环境变量PWD也动态包含立刻当前目录。

这个变量在脚本中会用的很多。

That’s it.

你学会了如何使用pwd命令了。当然这个命令非常简单，但在CLI下用的很多。-P选项其实也很少用，就算在shell脚本里面，人们应该也更倾向于使用readlink。
